# 第1课 揭秘 AI Agent

## 本节目标
- 理解什么是 AI Agent
- 理解 Agent 和聊天机器人的区别
- 理解 OpenClaw 的本地优势

## 1. 什么是 AI Agent
Agent 不是简单问答工具，而是能理解目标、规划步骤、调用工具并执行任务的系统。

## 2. Agent 和 Chatbot 的区别
- Chatbot 偏向单轮对话
- Agent 偏向目标执行
- Agent 可以连接工具和外部环境

## 3. OpenClaw 的本地优势
- 本地部署
- 隐私友好
- 可扩展连接器
- 可做个人 AI 中枢

## 总结
- Agent = 模型 + 工具 + 规划 + 执行
- OpenClaw 更适合做个人工作流助手
