# Project Roadmap

## Positioning

Build a Markdown-first open source tool that converts structured slide content into polished, single-file HTML presentations.

## Product Definition

- Input: `Markdown + images`
- Output: single-file `HTML`
- Primary user: content creators, educators, founders, and operators who want web-native decks without hand-writing frontend code

## Milestones

### M1

- Markdown convention
- Basic theme system
- Single-file HTML generation
- Keyboard, wheel, and touch navigation
- Mobile-friendly viewport fitting

### M2

- Better image and code layouts
- Cleaner auto-slide splitting
- Frontmatter support
- CLI entry

### M3

- Optional PDF/PPTX import flow
- Notes support
- Template inheritance
- AI-assisted structure suggestions
