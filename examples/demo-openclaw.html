样例：第 1 课如何拆成网页课件

课程主题

什么是 AI Agent

不是聊天窗口，而是“理解目标、规划步骤、调用工具、执行任务”的系统。

Agent vs Chatbot

Chatbot 偏单轮回答；Agent 更强调持续状态、环境交互和目标完成。

OpenClaw 的价值

本地优先、隐私友好、可接工具、可接工作流，适合做个人 AI 中枢。

对应课件页面建议

  • 第 1 页：封面 + 本节收获
  • 第 2 页：Agent 概念图
  • 第 3 页：Chatbot 与 Agent 对比
  • 第 4 页：OpenClaw 本地优势
  • 第 5 页：行动清单与下一课预告
这类内容特别适合用 Markdown 维护，再生成 HTML 3 / 5
